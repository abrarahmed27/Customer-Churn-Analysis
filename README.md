# Customer-Churn-Analysis
Predicted Customer Churn Analysis using PowerBi, SQL, Excel, Python, Machine learning for the telecom company equipping with enhanced visualization for best business decision process.
Salient Features:
* Segemented churning customers based on age, gender, payment plans and other demographics
* Identifying loyal customers and churners are pre-determined and action taken accordingly.

The telecom customer churn analysis utilized a holistic exploratory data analysis of the customer behavioral data in the telecom industry. This examination included creating usage and subscription features and developing predictive models for future customers that are likely to 'churn' using both Logistic Regression and Random Forest techniques in Python (Pandas, Scikit-learn). These activities ultimately supported business decisions that would help improve retention rates.
